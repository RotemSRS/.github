# RotemSRS (로템에스알에스)

로템에스알에스는 **철도 운영·유지보수(O&M) 전문 기업**으로, 운영 현장의 안전과 효율을 높이기 위한 **IT 플랫폼 기반의 업무 체계화**에 집중합니다.

## What we build (IT Platform for AX)
- 철도 운영·유지보수 업무의 **데이터 기반 표준화/자동화**(AX 전환)
- 운영/정비 현장의 의사결정을 돕는 **운영·정비 통합 정보 플랫폼**
- 노선/현장별 요구를 반영한 **MIS(경영정보)·운영지원 시스템** 고도화 

## Focus areas
- **Railway O&M Platform**: 운영·정비 프로세스, 실적/이력, 성과관리의 플랫폼화 
- **MIS / Operational Systems**: 신안산선 경영정보시스템(MIS) 구축/운영 경험 기반의 현장 최적화 
- **Global Maintenance Enablement**: 해외 유지보수 수행 경험을 기반으로 한 운영/정비 체계 확장 

## Why this GitHub Org exists
RotemSRS의 AX 전환을 위해, 본 Organization은
- 플랫폼/서비스 코드,
- 자동화 및 운영 도구,
- 문서화(표준/가이드/설계),
- PoC 및 레퍼런스 아키텍처
를 **일관된 품질과 재사용성**으로 관리하는 단일 작업 공간입니다.

## Collaboration
- Issues / PR 기반의 변경 관리
- 문서 우선(Architecture/Runbook) + 자동화 우선(CI/CD)
